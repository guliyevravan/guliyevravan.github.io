Bu gün sizə Hesablama Palatasının rəsmi veb səhifəsi haqqında məlumat verəcəm. Hesablama Palatası Milli Məclis tərəfindən yaradılan, Azərbaycanın Ali Audit Qurumudur. Saytın həm frontend (Vue) həm də backend (Nodejs) hissəsi mənim tərəfimdən hazırlanmışdır

Frontend

Sayt tamamilə SPA (Single Page Application) texnologiyasıdır. Saytda bootstrap ilə paralel olaraq bəzi hissələrdə native css də istifadə edilib.

İstifadə edilən modullar

  • Vue Router
  • Vuex
  • Axios

Backend

Backend RestfulAPI arxitekturasını təmin edir. Hər bir Request (sorğu) üçün router-lar bölünüb və hər biri üçün controller, modellər tətbiq olunub.

İstifadə edilən modullar

  • Express
  • Nodemailer
  • Multer
  • Jsonwebtoken
  • Bcryptjs
